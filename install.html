Install Access Rule Delayed Attempt quizaccess_delayed

Auto Activate the Quiz Attempt Button with a randomized delay to reduce the load at the start of the quizes.

Install from Moodle

The plugin is available in Moodle's catalog of plugins. You can install the latest version searching "delayed" or directly from the address https://moodle.org/plugins/quizaccess_delayed

Download the module

You can download it from the GitHub repository in ZIP format or by cloning the repository (recommended).
The branch "master" contains the most stable version of the plugin.
The "develop" branch has the most recent version with new features. This branch is just for developing and testing.


Install it in Moodle

Log in to your Moodle webpage as an administrator and visit the administration page. All the plugins to be updated or installed will appear, including the Quiz Delayed Attempt module. Click on the Update Moodle database button in order to proceed with the installation.

Configure the default settings

You can configure several parameters of the module:
- Enable/disable the Rule system-wide.
- Allow the teachers to use or not the rule in their quizzes.
- Set the default value for new quizzes.
- Set the target entry rate to a quiz.
- Maximum delay allowable (institution's policy).
- Maximum delay  as percentage of the quiz time-limit.
- Style of the countdown.
- Enable a system that warns the teacher if the quiz is configured with potentially problematic parameters.
- Define advices for the teachers and the students.
